Мини-мастер-класс по разработке веб-приложения. Попробуем вместе на ASP.NET реализовать приложение по регистрации на конференцию. Что сделаем:
- страницу регистрации на Razor Pages с привязкой к классу, куда будем собирать данные;
- модель данных для записей о регистрации;
- байндинг параметров POST-запроса на поля модели;
- напишем DbContext, сконфигурируем базу, сгенерируем и применим миграции;
- реализуем отображение списка участников, страницу подтверждения регистрации, сделаем на неё редирект;
- улучшим внешний вид наших страниц с помощью Bootstrap;
- реализуем валидацию вводимых данных на сервере и на клиенте (если успеем).

Приносите ноутбуки!
